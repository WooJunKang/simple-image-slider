# Simple Image Slider

React-Hooks를 이용한 간단한 이미지 슬라이더.
슬라이더는 크게 2개의 컴포넌트로 구성되어 있습니다. (`selector` & `slider`) <br/>
`selector`에서 선택된 이미지를 `slider`에서 보여줍니다. <br/>
추가된 이미지가 6개를 초과하게되면 좌우로 이동할 수 있는 버튼이 생성되고, 나머지 이미지를 확인할 수 있습니다.
